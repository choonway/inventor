However, it saves only the currenttransformation; all other state is left as is. This node can be useful for positioning a camera, since the transformations to the camera will not affect the rest of the scene,even through the camera will view the scene. Similarly, this node can be used to isolate transformations to light sources or other objects. METHODS SoTransformSeparator() Creates a transform separator node with default settings.
