SoRotation
NAME
SoRotation -- node representing a 3D rotation about an arbitrary axis
INHERITS FROM
SoBase > SoFieldContainer > SoNode > SoTransformation > SoRotation
DESCRIPTION
This node defines a 3D rotation about an arbitrary axis through the origin. The rotation is accumulated into the current transformation, which is applied tosubsequent shapes. The rotation field provides a variety of methods for specifying the rotation.
FIELDS
SoSFRotation rotation
Rotation specification.
