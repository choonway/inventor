SoPickedPointList
NAME
SoPickedPointList -- maintains a list of pointers to SoPickedPoint instances
INHERITS FROM
SbPList > SoPickedPointList
DESCRIPTION
This subclass of SbPList holds lists of pointers to instances of classes derived from SoPickedPoint. It is used primarily to return information from picking with theSoRayPickAction class.
