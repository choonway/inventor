SbBox2f

NAME
SbBox2f -- 2D box class
INHERITS FROM
SbBox2f
DESCRIPTION
2D box which has planes parallel to the major axes and is specified by two points (specified as floating point) on a diagonal. This class is part of the standard Inventordatatype classes and is used as input and output to geometry operations.
