NAME
SoDetail -- base class for describing detail information about a shape node
INHERITS FROM
SoDetail
DESCRIPTION
SoDetail is the abstract base class for all detail classes. A detail contains shapespecific information about a particular shape during picking and primitivegeneration. Subclasses store information based on the particular type of shape.
