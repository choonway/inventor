However, it also changes the transparency component. Thecolor and transparency information is packed into unsigned long integers: 0xaabbggrr, where aa represents the alpha (0x00 = fully transparent, 0xff = opaque),and bb, gg, and rr represent the blue, green, and red components of the color, respectively. SoPackedColor uses less memory than SoBaseColor or SoMaterial to store multiplecolor and transparency values. It can be used in cases where space is critical.
