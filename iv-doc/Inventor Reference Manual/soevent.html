SoEvent
NAME
SoEvent -- base class for all events
INHERITS FROM
SoEvent
DESCRIPTION
SoEvent is the base class for events in the Inventor event model. An event typically represents a user action, such as a mouse button being pressed or a keyboard keybeing released. SoEvent contains general information found in all Inventor events, including the time the event occurred, the position of the locater when the eventoccurred, and the state of the modifier keys when the event occurred.
