NAME
SoWriteAction -- Writes a scene graph to a file
INHERITS FROM
SoAction > SoWriteAction
DESCRIPTION
This class is used for writing scene graphs to files. It contains an SoOutput instance that by default writes to the standard output. Methods on the SoOutput can becalled to specify what file or memory buffer to write to.
