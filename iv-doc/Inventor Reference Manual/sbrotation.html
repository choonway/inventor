SbRotation
NAME
SbRotation -- class for representing a rotation
INHERITS FROM
SbRotation
DESCRIPTION
Object that stores a rotation. There are several ways to specify a rotation: quaternion (4 floats), 4x4 rotation matrix, or axis and angle. All angles are in radians and all rotations are right-handed.

Constructors for rotation. Matrix constructor requires a valid rotation matrix. Rotation constructor defines rotation that rotates from one vector into another. The rotateFrom and rotateTo vectors are normalized by the constructor before calculating the rotation.
