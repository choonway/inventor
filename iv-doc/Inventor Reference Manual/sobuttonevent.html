NAME
SoButtonEvent -- base class for all button events
INHERITS FROM
SoEvent > SoButtonEvent
DESCRIPTION
SoButtonEvent represents generic button press and release events in the Inventor event model. It is the base class for device-specific button events, namelySoKeyboardEvent, SoMouseButtonEvent, and SoSpaceballButtonEvent. This class stores the down/up state of the button when the event occurred.
