SoColorIndex
NAME
SoColorIndex -- surface color index node
INHERITS FROM
SoBase > SoFieldContainer > SoNode > SoColorIndex
DESCRIPTION
This node specifies the color to use for subsequent shapes as an index into the current color table. This is used only for BASE_COLOR lighting (see SoLightModel)in color index mode. Color index mode may be enabled by the window in which rendering occurs. Since color indices make sense only in the context of OpenGL rendering, this nodeimplements only a method for the SoGLRenderAction.
