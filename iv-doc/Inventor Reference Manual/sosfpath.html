SoSFPath
NAME
SoSFPath -- field containing a pointer to an SoPath
INHERITS FROM
SoField > SoSField > SoSFPath
DESCRIPTION
This field maintains a pointer to an SoPath instance, correctly maintaining its reference count. SoSFPaths are written to file as the path they point to. See the SoPath manual pagefor a description of the file format for a path.
