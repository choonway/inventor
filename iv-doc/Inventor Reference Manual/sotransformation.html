SoTransformation

NAME
SoTransformation -- Abstract base class for all geometric transformation nodes
INHERITS FROM
SoBase > SoFieldContainer > SoNode > SoTransformation
DESCRIPTION
This is the abstract base class for all nodes that perform geometric transformations.

It exists only to make it easy for applications to test whether a particular node is atransformation node (that is, is derived from this class).
