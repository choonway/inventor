SbBox2s

NAME
SbBox2s -- 2D box class
INHERITS FROM
SbBox2s
DESCRIPTION
2D box which has planes parallel to the major axes and is specified by two points(specified with short integers) on a diagonal. This class is part of the standardInventor datatype classes and is used as input and output to geometry operations.
