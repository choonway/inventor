SoUnits

NAME
SoUnits -- node that scales to convert units of length
INHERITS FROM
SoBase > SoFieldContainer > SoNode > SoTransformation > SoUnits
DESCRIPTION
This node defines a uniform 3D scale about the origin relative to the previously defined units. The default units for all data are meters. Adding a units node with thevalue INCHES will have the same effect as adding an SoScale node with the scaleFactor of (.0254, .0254, .0254). Any subsequent SoUnits node will take theprevious units into account. When building a composite object out of a bunch of pieces, it would be a good practice to add an SoUnits node at the beginning of eachof the pieces, under an SoSeparator node, to make sure all the pieces fit together with the same scale.
