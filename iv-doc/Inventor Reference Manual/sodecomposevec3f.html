SoDecomposeVec3f
NAME
SoDecomposeVec3f -- decomposes 3D vectors into floating-point values
INHERITS FROM
SoBase > SoFieldContainer > SoEngine > SoDecomposeVec3f
DESCRIPTION
This engine takes as input a 3D vector, and decomposes it into three single floatingpoint values. The input can have multiple values, allowing the engine to decompose severalvectors in parallel.
INPUTS
SoMFVec3f vector
Vector to be decomposed.
OUTPUTS
(SoMFFloat) x
First component of the vector.
SoMFFloat) y
Second component of the vector.
(SoMFFloat) z
Third component of the vector.
