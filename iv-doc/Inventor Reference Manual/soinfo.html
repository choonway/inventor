SoInfo
NAME
SoInfo -- node containing information text string
INHERITS FROM
SoBase > SoFieldContainer > SoNode > SoInfo
DESCRIPTION
This class defines a information node in the scene graph. This node has no effect during traversal. It is used to store information in the scene graph, typically forapplication-specific purposes, copyright messages, or other strings. This node differs from the SoLabel node in that it stores its information in an SbString instead of anSbName; the SbString is more efficient for storing long strings that don't have to be accessed very often. Use an SoLabel node for short strings that have to be accessed more often.
FIELDS
SoSFString string
Defines the info string value as an SbString.
