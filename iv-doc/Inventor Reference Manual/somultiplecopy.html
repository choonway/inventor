SoMultipleCopy

NAME
SoMultipleCopy -- group node that traverses multiple times, applying matrices
INHERITS FROM
SoBase > SoFieldContainer > SoNode > SoGroup > SoMultipleCopy
DESCRIPTION
This group node traverses its children, in order, several times, applying a different matrix each time. The matrices are stored in the multiple-value matrix field. Each matrix is concatenated to the current transformation matrix, and all of the children are traversed. This allows the user to put multiple copies of the same data indifferent locations easily and efficiently. Traversing the Nth child sets the current switch value to N, for use with inherited switch values (see SoSwitch).
FIELDS
SoMFMatrix matrix
Set of matrices to apply to children.
