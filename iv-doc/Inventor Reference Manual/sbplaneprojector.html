NAME SbPlaneProjector -- plane projector
INHERITS FROM
SbProjector > SbPlaneProjector
DESCRIPTION
SbPlaneProjector projects the mouse onto a plane. This is typically used to write interactive 3D manipulators and viewers.
METHODS
SbPlaneProjector(SbBool orient = FALSE)
Constructor. The default plane passes through the origin and is orientedperpendicular to the Z axis.

Constructor which is passed a plane. If orient to eye is TRUE, the plane willbe reoriented to the eye.

void setOrientToEye(SbBool orientToEye)
SbBool isOrientToEye() const
Set and get whether the projector should be oriented towards the eye. IforientToEye is set to TRUE, the given plane's direction is ignored, and the plane will be oriented to the eye. It will pass through the same point fromthe origin defined by the original dir * dist. Set to FALSE if the plane's direction should remain in working space.
